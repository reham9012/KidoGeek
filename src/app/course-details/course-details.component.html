Overview

Course Content

CodaKid Web Development delves into the foundations of web development using HTML, CSS, and Javascript. Projects in this track involve the ground up creation of your own custom website and understanding how to use browser tools to create them.

Upon successful completion of Web Development, CodaKid students will be ready to graduate to other CodaKid Core or Elective tracks.
